# exception117

In order to observe CWE ID 117 run the application and use following simple vector: http://localhost:8080/?dataFromRequest=data%0a%0ddata
It's somewhat difficult to resolve the problem assuming that one wants to log exceptions - recursive cleanup of Exception#getMessage would have to be put in place or even #toString? Could also be logger specific.
